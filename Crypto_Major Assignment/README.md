## 密码算法实现集合

📘 项目简介

包含多种经典及实用密码算法的集合，涵盖对称加密、非对称加密、密码分析破解及密码学辅助工具等。

🧮 实现算法

程序实现了以下密码学相关算法，涵盖加密、解密、攻击、检测等多个维度：

| 程序                 | 说明                                 |
|--------------------|------------------------------------|
| RSA                | 基于大数分解难题的非对称加密算法，支持数据加密、解密及数字签名    |
| SM2                | 中国国家密码标准非对称算法，支持密钥交换、数字签名及加密解密     |
| ElGamal            | 基于离散对数难题的非对称加密算法，支持加密及数字签名         |
| DES                | 经典对称分组加密算法，采用64位分组和56位密钥，实现数据加密解密  |
| DES线性攻击            | 针对DES算法的线性密码分析方法，通过线性近似式恢复密钥       |
| DES五轮差分            | 针对DES五轮加密的差分密码分析技术，利用差分特性破解密钥      |
| 维吉尼亚密码破解           | 针对维吉尼亚多表代换密码的破解工具，支持通过统计分析恢复密钥和明文  |
| Berlekamp-Massey算法 | 从输出序列中重构线性反馈移位寄存器（LFSR）最短连接多项式的核心算法 |
| 费马素性检测             | 概率性素性测试算法，快速验证一个数是否为素数，用于加密算法密钥生成  |
| CRT-share          | 基于中国剩余定理（CRT）的秘密共享算法，实现秘密的分存与恢复    |

⚙️ 环境依赖

- Python 3.8+（所有算法程序的运行基础环境）

- numpy（DES线性攻击、DES五轮差分、Berlekamp-Massey算法、费马素性检测）

- pycryptodome（RSA、SM2、ElGamal、DES算法）

- gmpy2（RSA、SM2、ElGamal、费马素性检测算法、CRT-share算法）


📚 部分算法原理简介

**DES五轮差分攻击**：作为差分密码分析的经典应用，核心是利用DES算法S盒（ substitutions盒）的差分特性。首先定义明文对的“输入差分”（特定比特位的差异模式），通过5轮DES加密后，统计“输出差分”的出现概率-若某输出差分出现概率显著高于随机情况，即可建立输入差分与密钥比特的关联。 

攻击时，先构造大量符合目标输入差分的明文对并获取对应密文对，通过统计密文对的差分分布筛选候选密钥，逐步缩小范围直至恢复正确密钥，该方法针对性突破了DES的轮加密结构。

**维吉尼亚密码破解**：围绕“多表代换的周期性”展开，关键步骤分三步。

第一步是确定密钥长度：通过“卡方检验”或“重合指数法”分析密文——随机文本的重合指数（字母出现频率的统计特征）与自然语言存在固定差异，当密文按某一长度m分组后，若每组重合指数接近自然语言，m即为密钥长度。

第二步是恢复密钥：将密文按密钥长度分组，每组对应密钥的一个字符，此时每组可视为单表代换密码（如凯撒密码），通过统计每组内字母出现频率，与自然语言字母频率（如英文中‘e’出现频率最高）比对，推算每组对应的密钥字符。

第三步是验证与修正：用推测密钥解密密文，根据语义通顺度调整密钥，最终得到正确明文与密钥。

**Fermat素性检测算法**：一种经典的概率性素性测试算法，核心基于费马小定理——若p为素数，且a与p互质，则a^(p-1) ≡ 1 mod p。

算法步骤为：首先排除p=2以下的非素数情况，然后随机选取若干个与p互质的底数a，计算a^(p-1) mod p的值，若结果均为1，则p大概率为素数；若存在任一a使得结果不为1，则p确定为合数。

该算法优势是运算速度快，适合大整数的初步素性筛选，为RSA、SM2等非对称算法的密钥生成（需大素数）提供高效支撑，但存在“卡迈克尔数”这类伪素数可能导致误判，实际应用中常与其他检测算法结合使用以提升准确性。
